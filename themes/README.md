# Community Themes

### How to share
1. Open the preferences in any card.
2. Go to the preset menu (menu next to the Save button).
3. Click **Share Theme** and save the `.json` file to your computer.
4. Upload that file to this folder via a Pull Request, or open an Issue with the file.

### How to use
1. Download any `.json` file from this folder.
2. Open the preferences in Anki.
3. Open the preset menu and click **Import Theme**.
4. Select the `.json` file you downloaded.
