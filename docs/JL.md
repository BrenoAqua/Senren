# Setup: JL

Configure JL to automatically populate Senren's fields when creating cards.

## Fields Configuration:

In JL Preferencces -> Anki:

| Field | Selected Value |
| :--- | :--- |
| **word** | Primary Spelling |
| **sentence** | Sentence |
| **sentenceFurigana** | Nothing |
| **sentenceEng** | Nothing |
| **reading** | Readings |
| **sentenceCard** | Nothing |
| **audioCard** | Nothing |
| **notes** | Nothing |
| **selectionText** | Selected Definitions |
| **definition** | Definitions |
| **wordAudio** | Audio |
| **sentenceAudio** | Nothing |
| **picture** | Image (Clipboard) |
| **glossary** | Nothing |
| **pitchPosition** | Pitch Accents (Numeric) |
| **pitch** | Nothing |
| **frequency** | Frequencies |
| **freqSort** | Frequency (Harmonic Mean) |
| **miscInfo** | Nothing |
| **dictionaryPreference** | Nothing |
